What is HTML and Why Do We Care

What is HTML?

HTML, standing for HyperText Markup Language, is a type of markup language.

A markup language is a system for annotating a document in a way that is syntactically distinguishable from the text. ---- From Wikipedia

In another words, a markup language is essentially a text script with parts that are used to indicate forms, styles or transformation of normal text parts, instead of simply functioning as normal texts.

If we look at an HTML file, we see that some parts of the script is just normal text but there are also parts such as all the tags that are surrounded by angle brackets.

Is there other type of Markup Languages?

Oh, definitely! There are famous others like XML(Extensible Markup Language), XHTML(Extensible HyperText Markup Language), and one very important one that you will or you already have run into, the TeX family (most famous one LaTex).

So why do we care about HTML

A simple answer would be: because this is the core language of web development.

The purpose of a web browser (Chrome, IE, Firefox, Safari) is to read HTML(sometimes XHTML) documents and interpret them into nice(or shaddy) webpages.

About Some Combined Tags

These tags have their own name but actually they are just the built-in tags combined by different formatting.

Examples: Quotations, abbreviation, citation, address and so on...

If you are interested, just look them up.

One thing might be fairly interesting is <bdo>,bidirectional override, which is used to define the direction of text. left to right and right to left. This can be used for websites for languages with different text directions, such as Arabic.
